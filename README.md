Crowd4U 2.0 書誌同定タスク Redirect
===================================

書誌同定タスクのリダイレクトを行うリポジトリ

- Crowd4U 2.0 が何らかのカタチでエラーになった際に、容易にリダイレクト先変更できるようにすることが目的

- `docs/index.html` の `WORKFLOW_ID` を書き換えることで、ワークフロー(リダイレクト先)を変更可能
